# hotel_booking-dataset
# Hotel Booking Analysis using SQL and Power BI

## Project Overview
The Hotel Booking Analysis project focuses on analyzing hotel booking data to understand customer behavior, booking trends, cancellation patterns, and revenue generation. This project uses Excel for data cleaning, MySQL for data analysis, and Power BI for creating interactive dashboards.

## Objectives
- Analyze hotel booking trends
- Study customer behavior
- Identify cancellation patterns
- Generate revenue insights
- Create interactive dashboards
- Support business decision-making

## Tools Used
- Excel – Data Cleaning
- MySQL – Data Analysis
- Power BI – Dashboard Visualization
- GitHub – Project Storage

## Dataset
Dataset Source: Kaggle – Hotel Booking Dataset

Dataset includes:
- Hotel Type
- Lead Time
- Arrival Date
- Customer Type
- Country
- ADR (Average Daily Rate)
- Cancellation Status
- Reservation Status

## Project Workflow

### Step 1: Data Collection
Downloaded the Hotel Booking Dataset from Kaggle.

### Step 2: Data Cleaning
- Removed duplicate records
- Handled missing values
- Formatted data fields
- Checked consistency

### Step 3: SQL Analysis
Performed:
- Total bookings analysis
- Revenue analysis
- Customer type analysis
- Cancellation analysis
- Country-wise booking analysis

### Step 4: Power BI Dashboard
Created:
- KPI Cards
- Revenue Charts
- Booking Trend Analysis
- Customer Analysis
- Interactive Filters

## Dashboard Features
- Total Bookings KPI
- Revenue KPI
- Average Price KPI
- Cancellation KPI
- Country Analysis
- Customer Analysis
- Booking Trend Analysis

## Business Insights
- City hotels generate higher revenue
- High lead time increases cancellations
- Customer behavior affects booking trends
- Seasonal trends influence bookings
- Dynamic pricing improves business performance
